## Description
This dataset provides a curated glimpse into the world of hip-hop, featuring tracks and artists that have significantly influenced the genre's landscape. It includes around 440 entries of the most celebrated hip-hop tracks and artists as per Spotify's music public API.

## Source
The data was obtained from Kaggle's dataset repository and is based on Spotify's public API.

## Contents
- Artist: The name of the artist.
- Track Name: The title of the track.
- Popularity: A numeric score reflecting the track's reception and appeal among Spotify listeners.
- Duration (ms): The track's length in milliseconds.
- Track ID: A unique identifier within Spotify's ecosystem.
